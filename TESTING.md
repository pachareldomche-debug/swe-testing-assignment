# Testing Strategy

The project uses unit tests and integration tests.

Unit tests verify individual calculator functions such as addition, subtraction, multiplication and division.

Integration tests verify that different parts of the application work together.

# Testing Pyramid

Most tests are unit tests while only a few are integration tests.

# Black Box vs White Box

Unit tests use white-box testing because they directly test the internal logic.

Integration tests use black-box testing because they simulate user behaviour.

# Regression Testing

The test suite can be used to detect regressions when the code is modified in the future.
